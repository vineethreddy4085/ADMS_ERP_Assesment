# Item Processing App

## Description
This is a simple ASP.NET MVC application where items can be created and processed into child items.

## Features
- Add items with weight
- Process items (create child items)
- Display parent-child hierarchy

## Technologies
- ASP.NET MVC
- C#
- SQL Server (script included)

## How to Run
1. Open project in Visual Studio
2. Build and run
3. Use forms to add and process items

## Database
Run the SQL script in SQL Server to create the Items table.
